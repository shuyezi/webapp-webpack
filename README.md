## 关于
	
	# 功能
		· 项目打包，javascript、style、images压缩合并 ( release文件夹为最终发布目录 )。
		· 成分：ES6、React、Webpack
		· 配置环境：./.env.js

	# 使用
		· 克隆项目 clone xxx
		· 安装依赖包 nmp install
		· 启动项目（配置./.env.js为相应环境：debug/test/production）
			· @开发环境：npm run dev
			· @测试环境 && @生产环境：npm run build；分别生产对于的发布目录

	# 目录
